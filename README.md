## Tech Stack Used

- Bakcend Framework: SpringBoot
- Backend Language: Java
- Packaging: Apache Maven
- Containerisation: Docker
- Backend Hosting: Render
- Structure: HTML
- Logic: JavaScript
- Styling: CSS
- Frontend Hosting: Github Pages
